# Cheatsheets

Purpose
-------
One-page quick references for common CLI patterns and commands you will use during interviews and day-to-day tasks.

Examples
--------
- `git-cheatsheet.md` — common git workflows and commands
- `docker-cheatsheet.md` — common docker build/run/debug commands
- `kubectl-cheatsheet.md` — common kubectl commands and debugging tips
- `terraform-cheatsheet.md` — frequently used Terraform CLI patterns

How to add a cheatsheet
-----------------------
Create a new file named `{topic}-cheatsheet.md` and keep it concise (no more than 1-2 pages). Use examples and short commands.
