# Git Cheatsheet

Common commands:

- Clone: git clone <repo>
- Status: git status
- Add: git add <file>
- Commit: git commit -m "msg"
- Branch: git branch
- Create branch: git checkout -b feature/x
- Merge: git checkout main; git merge feature/x
- Rebase interactive: git rebase -i <commit>
- Stash: git stash save "msg"; git stash pop
- Undo last commit: git reset --soft HEAD~1
