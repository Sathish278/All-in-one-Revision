# Networking - Consolidated Revision

Short, practical networking review for DevOps.

## Topics
- OSI model, TCP vs UDP, routing, switching
- Subnetting (CIDR), NAT, DHCP, DNS
- VLANs, trunk vs access ports, firewall zones
- VPN basics and common protocols (IPSec, OpenVPN)

## Quick Commands
- Test DNS: nslookup example.com
- Test connectivity: ping 8.8.8.8; traceroute 8.8.8.8
- Show interfaces: ip a; ifconfig -a
- Show routing table: ip route
